개인정보처리방침

📱 워킹 머니 트래커는 사용자의 개인정보를 수집하지 않는 앱입니다.

1. 개인정보 수집 현황

워킹 머니 트래커는 다음과 같은 개인정보 보호 정책을 유지합니다:

  • ✅ 개인정보 수집 없음: 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다
  • ✅ 계정 생성 불필요: 별도의 회원가입이나 로그인이 필요하지 않습니다
  • ✅ 외부 전송 없음: 입력된 데이터가 외부 서버로 전송되지 않습니다

2. 데이터 저장 및 관리

앱에서 사용하는 모든 데이터는 사용자 기기에만 저장됩니다:

  • 🏠 로컬 저장: 시급, 월급, 출퇴근 시간 등의 정보는 사용자의 iPhone에만 저장
  • 🔒 앱 내부 저장소: iOS의 안전한 앱 샌드박스 내에서만 데이터 관리
  • 🚫 클라우드 동기화 없음: iCloud나 다른 클라우드 서비스와 동기화되지 않음
  • 🗑️ 앱 삭제 시 완전 제거: 앱을 삭제하면 모든 데이터가 함께 삭제됨

3. 입력 정보 종류

사용자가 입력할 수 있는 정보는 다음과 같으며, 모두 기기 내부에만 저장됩니다:

  • 💰 시급 또는 월급 정보
  • ⏰ 출근/퇴근 시간
  • 🍽️ 점심시간 차감 설정
  • ⚙️ 앱 사용 설정 (위젯 모드 등)

4. 권한 사용

워킹 머니 트래커는 최소한의 권한만을 사용합니다:

  • 📱 필수 권한: 기본 앱 실행 권한
  • ⏰ 선택적 권한: 화면 항상 켜짐 (배터리 소모 방지용, 거부 가능)
  • 🚫 불필요한 권한: 카메라, 위치, 연락처, 사진 등의 권한은 요청하지 않음

5. 데이터 보안

사용자 데이터의 안전성을 위해 다음과 같은 조치를 취합니다:

  • 🔐 iOS 기본 보안 기능 활용 (앱 샌드박스, 데이터 암호화)
  • 🌐 네트워크 통신 없음 (데이터 유출 위험 차단)
  • 🛡️ 제3자 분석 도구 미사용

6. 정책 변경

본 개인정보처리방침에 변경이 있을 경우:

  • 📝 앱 업데이트를 통해 사용자에게 알림
  • 🔄 중대한 변경 시 사용자 동의 재요청
  • 📅 변경 사항은 본 페이지에 즉시 반영
